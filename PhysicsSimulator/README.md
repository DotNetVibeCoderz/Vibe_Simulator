# Physics Simulator

A fun and interactive Physics Simulator built with **C#** and **Avalonia UI**.
Created by **Jacky The Code Bender** (Gravicode Studios).

![Avalonia](https://img.shields.io/badge/Avalonia-11.0-purple) ![NET](https://img.shields.io/badge/.NET-8.0-blue)

## Features (Simulations)

1.  **Rigid Body Collisions**: Bouncing balls with elastic collisions.
2.  **Pendulum & Springs**: Double pendulum chaos trace.
3.  **Fluid Dynamics**: Simple particle-based water simulation.
4.  **Gravity & Orbits**: N-Body simulation (Solar system builder).
5.  **Wave Interference**: Ripple tank visualization.
6.  **Crowd Simulation**: Boids algorithm (Flocking).

## How to Run

1.  Install .NET 8.0 SDK.
2.  Run the command:
    ```bash
    dotnet run
    ```
3.  Enjoy!

## Controls

*   **Select Simulation**: Use the sidebar.
*   **Interaction**: Click or Drag on the canvas (depends on simulation).
*   **Reset**: Click the Reset button.

1.  Pastikan .NET 8.0 SDK terinstall.

