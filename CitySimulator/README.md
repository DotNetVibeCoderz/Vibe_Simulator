# CitySimulator

## English
CitySimulator is a web-based 3D City Simulation built using Blazor Server and Three.js. It features automatic zoning, dynamic weather (sun, rain, fog), day/night cycle, interactive camera controls (pan, zoom, orbit), and basic simulation mechanics including population growth, funds, and happiness.

**Features:**
- Real-time 3D rendering with level of detail (LOD) and instancing for optimal performance.
- Interactive User Interface HUD showing population, happiness, and funds.
- Automatic resource and road generation visualization.
- AI agents simulation (simple moving vehicles).
- Day/night cycle with dynamic lighting and shadows.
- Built entirely on C# Blazor Server and pure Three.js without external heavy assets to ensure fast load times.

**How to run:**
1. Open terminal and navigate to this project folder.
2. Run `dotnet run` or `dotnet watch run`.
3. Open the provided localhost URL in your browser.
4. Navigate via mouse and keyboard: Left-click and drag to orbit, right-click and drag to pan, scroll to zoom.

## Bahasa Indonesia
CitySimulator adalah Simulasi Kota 3D berbasis web yang dibangun menggunakan Blazor Server dan Three.js. Proyek ini dilengkapi dengan fitur zonasi otomatis, cuaca dinamis (cerah, hujan, kabut), siklus siang/malam, kontrol kamera interaktif (pan, zoom, orbit), dan mekanika simulasi dasar seperti pertumbuhan populasi, dana, dan tingkat kebahagiaan.

**Fitur:**
- Rendering 3D waktu nyata dengan *level of detail* (LOD) dan instancing untuk performa yang optimal.
- Antarmuka pengguna (HUD) interaktif yang menampilkan populasi, kebahagiaan, dan dana.
- Visualisasi pembuatan sumber daya dan jalan secara otomatis.
- Simulasi agen AI (kendaraan yang bergerak sederhana).
- Siklus siang/malam dengan pencahayaan dan bayangan dinamis.
- Dibangun sepenuhnya di atas C# Blazor Server dan Three.js murni tanpa aset berat eksternal untuk memastikan waktu muat yang cepat.

**Cara Menjalankan:**
1. Buka terminal dan arahkan ke folder proyek ini.
2. Jalankan perintah `dotnet run` atau `dotnet watch run`.
3. Buka URL localhost yang muncul di browser Anda.
4. Navigasi dengan mouse dan keyboard: Klik kiri dan geser untuk orbit, klik kanan dan geser untuk pan, gulir (scroll) untuk zoom.
