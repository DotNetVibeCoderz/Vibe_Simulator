# Superman Simulator

A web-based 3D flight simulator inspired by Superman, built using Blazor Server and Three.js with a striking brutalist design. Experience free flight over a low-poly earth, take on rescue missions, and feel the thrill of supersonic speeds!

## Features

- **Global 3D Flight**: Fly freely over a procedurally generated brutalist landscape.
- **City Tour**: Weave through towering monolithic structures.
- **Dynamic Speed**: Control your acceleration from a gentle hover to blistering supersonic speeds.
- **Missions**: Accept challenges around the world (e.g., Earthquake in Tokyo, Eiffel Tower photo op).
- **Audio Simulation**: Dynamic wind noise based on your flight velocity.
- **Brutalist UI**: High-contrast, bold, and raw user interface.

## Tech Stack
- **C# / .NET 8** (Blazor Server)
- **JavaScript** (Three.js for WebGL rendering)
- **CSS** (Custom Brutalist styling)

## Controls
- **W / S**: Accelerate / Decelerate
- **A / D**: Yaw (Turn) Left / Right
- **Q / E**: Roll Left / Right
- **Space**: Pitch Up
- **Shift**: Turbo Boost (Hold)

# Simulator Superman (Bahasa Indonesia)

Simulator penerbangan 3D berbasis web yang terinspirasi oleh Superman, dibangun menggunakan Blazor Server dan Three.js dengan desain bergaya brutalisme yang mencolok. Rasakan kebebasan terbang di atas bumi *low-poly*, jalankan misi penyelamatan, dan rasakan sensasi kecepatan supersonik!

## Fitur Utama

- **Penerbangan Global 3D**: Terbang bebas melintasi lanskap brutalisme yang dibuat secara prosedural.
- **Tur Kota**: Meluncur melewati gedung-gedung monolitik raksasa.
- **Kecepatan Dinamis**: Atur akselerasi dari sekadar melayang hingga kecepatan supersonik yang membelah udara.
- **Misi**: Terima tantangan di berbagai belahan dunia (mis. Gempa di Tokyo, Sesi Foto di Menara Eiffel).
- **Simulasi Audio**: Suara angin dinamis berdasarkan kecepatan terbang Anda.
- **UI Brutalis**: Antarmuka pengguna yang kontras tinggi, tebal, dan mentah.

## Teknologi yang Digunakan
- **C# / .NET 8** (Blazor Server)
- **JavaScript** (Three.js untuk render WebGL)
- **CSS** (Gaya Kustom Brutalisme)

## Kontrol
- **W / S**: Akselerasi / Deselerasi
- **A / D**: Yaw (Belok) Kiri / Kanan
- **Q / E**: Roll Kiri / Kanan
- **Spasi**: Pitch Atas (Mendongak)
- **Shift**: Turbo Boost (Tahan)
