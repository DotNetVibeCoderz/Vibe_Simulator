# GBPlayer

GBPlayer is a GameBoy emulator built with C# and Avalonia UI. It provides a modern, cross-platform interface for playing your favorite GameBoy games.

## Features

- **Platform Support:** Runs on Windows, macOS, and Linux (via Avalonia UI).
- **Format Support:** Supports `.gb`, `.gbc`, `.bin`, `.gba` files.
- **UI:** Modern interface with menus for easy navigation.
- **Controls:** Keyboard support (configurable in future updates).
- **Display:** Resizable window with high-quality scaling.

## Requirements

- .NET 6.0 SDK or later.

## Installation

1. Clone the repository.
2. Open the solution in Visual Studio or JetBrains Rider.
3. Build and Run.

## Usage

1. Launch GBPlayer.
2. Click **File -> Load ROM...**
3. Select a GameBoy ROM file.
4. The game will start automatically.

## Controls

| Key | Action |
| --- | --- |
| Z | A Button |
| X | B Button |
| Enter | Start |
| Right Shift | Select |
| Arrow Keys | D-Pad |

## Credits

- **UI:** Avalonia UI
