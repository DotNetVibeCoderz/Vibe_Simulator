# Planet Simulator

## English
**Planet Simulator** is a 3D web application built with **Blazor (C#)** and **Three.js**. It allows users to visualize and interact with a procedural planet in real-time.

### Features
- **3D Rendering**: High-performance 3D rendering using Three.js.
- **Interactive Control Panel**: Change planet size, rotation speed, and color dynamically.
- **Advanced Effects**: Atmosphere glow, star field background, and dynamic lighting.
- **Modern UI**: Clean and responsive overlay controls.

### How to Run
1. Ensure you have the **.NET SDK** installed.
2. Open a terminal in the project folder.
3. Run the command:
   ```bash
   dotnet run
   ```
4. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5xxx`).
