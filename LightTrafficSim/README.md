# LightTrafficSim

## Description (English)
LightTrafficSim is a visual traffic simulation built with .NET (C#) and Avalonia UI. It demonstrates a simple one-way street traffic flow controlled by a traffic light.

### Features
- **Visual Simulation**: Watch cars spawn and move across the screen.
- **Traffic Light Logic**: Standard Red -> Green -> Yellow cycle.
- **Physics**: Cars respect the traffic light (stop at red) and maintain safe distance from cars ahead.
- **Customizable**: Adjust Spawn Rate and Traffic Light Durations in real-time using the control panel.
- **Cross-Platform**: Runs on Windows, Linux, and macOS.

### How to Run
1. Ensure you have the .NET 8.0 SDK installed.
2. Clone the repository.
3. Open a terminal in the project folder.
4. Run `dotnet run`.

## Deskripsi (Bahasa Indonesia)
LightTrafficSim adalah simulasi lalu lintas visual yang dibuat dengan .NET (C#) dan Avalonia UI. Aplikasi ini mendemonstrasikan arus lalu lintas jalan satu arah yang dikendalikan oleh lampu lalu lintas.

### Fitur
- **Simulasi Visual**: Lihat mobil muncul dan bergerak melintasi layar.
- **Logika Lampu Lalu Lintas**: Siklus standar Merah -> Hijau -> Kuning.
- **Fisika Sederhana**: Mobil mematuhi lampu lalu lintas (berhenti saat merah) dan menjaga jarak aman dari mobil di depan.
- **Dapat Disesuaikan**: Atur Tingkat Kemunculan Mobil dan Durasi Lampu secara real-time menggunakan panel kontrol.
- **Lintas Platform**: Dapat berjalan di Windows, Linux, dan macOS.

### Cara Menjalankan
1. Pastikan Anda telah menginstal .NET 8.0 SDK.
2. Clone repositori ini.
3. Buka terminal di folder proyek.
4. Jalankan perintah `dotnet run`.
