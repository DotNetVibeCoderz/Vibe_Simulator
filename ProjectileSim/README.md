# ProjectileSim

ProjectileSim is a modern, interactive projectile motion simulator built with C# and Avalonia UI. It allows students and educators to explore the physics of projectile motion with real-time visualization and data analysis.

## Features

- **Adjustable Launch Parameters:** Modify initial velocity, launch angle, and height.
- **Trajectory Visualization:** Real-time parabolic curve display.
- **Live Motion Statistics:** Monitor speed, position, time, range, and max height.
- **Energy Analysis:** Track Kinetic and Potential Energy changes.
- **Environment Control:** Adjust gravity, mass, and air resistance (drag).

## How to Run

1. Ensure you have the .NET 8.0 SDK installed.
2. Navigate to the project folder.
3. Run the following command:
   ```bash
   dotnet run
   ```

## Controls

- Use the sliders on the left panel to adjust parameters.
- Click **Start** to begin the simulation.
- Click **Pause** to freeze the motion.
- Click **Reset** to clear the trajectory and reset parameters.

# ProjectileSim (Bahasa Indonesia)

ProjectileSim adalah simulator gerak proyektil interaktif modern yang dibuat dengan C# dan Avalonia UI. Aplikasi ini memungkinkan siswa dan pengajar untuk mengeksplorasi fisika gerak parabola dengan visualisasi real-time dan analisis data.

## Fitur

- **Parameter Peluncuran yang Dapat Disesuaikan:** Ubah kecepatan awal, sudut peluncuran, dan ketinggian.
- **Visualisasi Lintasan:** Tampilan kurva parabola secara real-time.
- **Statistik Gerak Langsung:** Pantau kecepatan, posisi, waktu, jangkauan, dan ketinggian maksimum.
- **Analisis Energi:** Lacak perubahan Energi Kinetik dan Potensial.
- **Kontrol Lingkungan:** Sesuaikan gravitasi, massa, dan hambatan udara (drag).

## Cara Menjalankan

1. Pastikan Anda telah menginstal .NET 8.0 SDK.
2. Buka folder proyek.
3. Jalankan perintah berikut:
   ```bash
   dotnet run
   ```

## Kontrol

- Gunakan slider di panel kiri untuk menyesuaikan parameter.
- Klik **Start** untuk memulai simulasi.
- Klik **Pause** untuk membekukan gerakan.
- Klik **Reset** untuk menghapus lintasan dan mengatur ulang parameter.
