**BouncingBalls** adalah aplikasi simulasi fisika sederhana yang dibuat menggunakan C# dan Windows Forms. Aplikasi ini menampilkan bola-bola yang memantul di dalam sebuah poligon yang berputar.

### Fitur Utama
*   **Poligon Berputar**: Wadah utama berupa poligon yang berputar terus menerus.
*   **Fisika Tumbukan**: Simulasi tumbukan antar bola dan tumbukan antara bola dengan dinding poligon.
*   **Kustomisasi**:
    *   Ubah jumlah sisi poligon (Segitiga, Kotak, Pentagon, Hexagon, dst).
    *   Atur kecepatan putaran poligon.
    *   Tambah atau kurangi jumlah bola secara real-time.
*   **Visual**: Render grafis yang halus dengan anti-aliasing.

### Cara Menjalankan
1.  Pastikan Anda telah menginstal **.NET SDK**.
2.  Buka terminal atau command prompt di folder project.
3.  Jalankan perintah berikut:
    ```bash
    dotnet run
    ```

### Kontrol Aplikasi
*   **Balls**: Mengatur jumlah bola yang ada di layar. Gunakan tombol `+` atau `-` atau ketik angkanya langsung.
*   **Polygon Sides**: Mengatur jumlah sisi poligon (min: 3, max: 10).
*   **Rotation Speed**: Mengatur kecepatan putar poligon dalam derajat per detik.

**BouncingBalls** is a simple physics simulation application built with C# and Windows Forms. It demonstrates balls bouncing inside a rotating polygon container.

### Key Features
*   **Rotating Polygon**: The main container is a continuously rotating polygon.
*   **Collision Physics**: Simulates collisions between balls and collisions between balls and the polygon walls.
*   **Customization**:
    *   Change the number of polygon sides (Triangle, Square, Pentagon, Hexagon, etc.).
    *   Adjust the polygon's rotation speed.
    *   Add or remove balls in real-time.
*   **Visuals**: Smooth graphical rendering with anti-aliasing.

### How to Run
1.  Ensure you have **.NET SDK** installed.
2.  Open a terminal or command prompt in the project folder.
3.  Run the following command:
    ```bash
    dotnet run
    ```

### Application Controls
*   **Balls**: Adjust the number of balls on the screen. Use the `+` / `-` buttons or type the number directly.
*   **Polygon Sides**: Set the number of sides for the polygon (min: 3, max: 10).
*   **Rotation Speed**: Set the rotation speed of the polygon in degrees per second.
