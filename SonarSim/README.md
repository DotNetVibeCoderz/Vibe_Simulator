# SonarSim - Advanced Sonar Simulator

![SonarSim](https://via.placeholder.com/150)

## Deskripsi (Bahasa Indonesia)
SonarSim adalah aplikasi simulasi sonar desktop yang dibuat menggunakan C# dan Windows Forms. Aplikasi ini mensimulasikan deteksi objek bawah laut seperti penyelam, ikan, kapal, dan torpedo dengan tampilan grafis radar yang realistis.

### Fitur Utama:
1.  **Visualisasi Sonar Realistis**: Efek sapuan radar (sweep), fading blips, dan jejak pergerakan (trails).
2.  **Objek Dinamis**: Berbagai tipe objek (Penyelam, Ikan, Kapal, Torpedo) dengan kecepatan dan arah acak.
3.  **Alarm Perimeter**: Zona aman yang dapat diatur radiusnya. Alarm akan berbunyi (visual merah) jika objek asing masuk.
4.  **Sistem Tracking**: Tabel data real-time yang menampilkan ID, Tipe, Kecepatan, Arah, dan Status (Mendekat/Menjauh) dari objek yang terdeteksi.
5.  **Kontrol Simulasi**: Pengaturan kecepatan simulasi dan tombol Reset.
6.  **Objek Statis**: Bebatuan dan karang yang diam di dasar laut.

### Cara Menggunakan:
1.  Jalankan aplikasi `SonarSim.exe`.
2.  Gunakan **TrackBar Speed** untuk mempercepat atau memperlambat simulasi.
3.  Gunakan **TrackBar Perimeter** untuk mengatur jarak aman (lingkaran merah putus-putus).
4.  Pantau tabel di sebelah kanan untuk melihat detail objek yang tertangkap radar.
5.  Tekan tombol **Reset** untuk memulai ulang simulasi dengan posisi acak baru.

## Description (English)
SonarSim is a desktop sonar simulation application built using C# and Windows Forms. It simulates the detection of underwater objects such as divers, fish, ships, and torpedoes with realistic radar graphics.

### Key Features:
1.  **Realistic Sonar Visualization**: Radar sweep effects, fading blips, and movement trails.
2.  **Dynamic Objects**: Various object types (Diver, Fish, Ship, Torpedo) with random speeds and directions.
3.  **Perimeter Alarm**: Adjustable safety zone. An alarm triggers (visual red alert) if a foreign object enters the perimeter.
4.  **Tracking System**: Real-time data table showing ID, Type, Speed, Heading, and Status (Approaching/Receding) of detected objects.
5.  **Simulation Controls**: Simulation speed adjustment and Reset button.
6.  **Static Objects**: Rocks and reefs that remain stationary.

### How to Use:
1.  Run the `SonarSim.exe` application.
2.  Use the **Speed TrackBar** to speed up or slow down the simulation.
3.  Use the **Perimeter TrackBar** to adjust the safety distance (dashed red circle).
4.  Monitor the table on the right to see details of objects caught by the radar.
5.  Press the **Reset** button to restart the simulation with new random positions.
