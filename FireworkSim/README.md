[English]

A simple and interactive Firework Simulator built with C# and Windows Forms. This project demonstrates basic 2D physics simulations, particle systems, and GDI+ drawing operations.

## Features
- **Click to Launch**: Click anywhere on the black canvas to launch a firework from the bottom towards your mouse cursor's horizontal position.
- **Auto Fire Mode**: Sit back and watch! Automatically launches random fireworks when enabled.
- **Real-time Customization**: Use the control panel at the bottom to adjust simulation parameters instantly:
    - **Gravity**: Adjust how fast particles fall after explosion.
    - **Explosion Force**: Control the radius and power of the burst.
    - **Particle Count**: Set the density of the explosion (more particles = more CPU usage).
- **Vibrant Colors**: Uses HSL to RGB color conversion to generate bright and varied particle colors.
- **Physics Engine**: Implements basic kinematics including velocity, acceleration (gravity), and drag (air resistance).

## How to Run
1. Ensure you have the .NET SDK installed.
2. Open the solution/project in Visual Studio or VS Code.
3. Build the project.
4. Run the application (`F5` in Visual Studio).

## Controls
- **Left Click**: Launch a firework.
- **Control Panel**: Located at the bottom of the window to change settings.

Sebuah simulator kembang api sederhana dan interaktif yang dibuat dengan C# dan Windows Forms. Proyek ini mendemonstrasikan simulasi fisika 2D dasar, sistem partikel, dan operasi penggambaran GDI+.

## Fitur
- **Klik untuk Meluncurkan**: Klik di mana saja pada layar hitam untuk meluncurkan kembang api dari bawah menuju posisi horizontal kursor mouse Anda.
- **Mode Auto Fire**: Duduk dan nikmati! Meluncurkan kembang api secara otomatis dan acak saat diaktifkan.
- **Kustomisasi Real-time**: Gunakan panel kontrol di bagian bawah untuk menyesuaikan parameter simulasi secara instan:
    - **Gravitasi**: Atur seberapa cepat partikel jatuh setelah ledakan.
    - **Kekuatan Ledakan**: Kontrol radius dan kekuatan ledakan.
    - **Jumlah Partikel**: Atur kepadatan ledakan (lebih banyak partikel = penggunaan CPU lebih tinggi).
- **Warna Cerah**: Menggunakan konversi warna HSL ke RGB untuk menghasilkan warna partikel yang cerah dan bervariasi.
- **Mesin Fisika**: Mengimplementasikan kinematika dasar termasuk kecepatan, percepatan (gravitasi), dan hambatan udara (drag).

4. Jalankan aplikasi (`F5` di Visual Studio).
